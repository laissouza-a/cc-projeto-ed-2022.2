# Projeto
Matéria: Estrutura de Dados 2022.2

## Pré-requisitos

Sistema operacioanl: Ubuntu 22.04

Instalar Opengl
```bash
sudo chmod 777 install.sh
./install.sh
```

Executar o programa
```bash
 make all
```
